
win32exts for VBScript/JavaScript 说明：

（1）COM组件的选择：

win32exts 不仅同时提供了 32位（win32exts_dll.dll）和64位（win32exts_dll64.dll）的 进程内组件，
   还同时提供了 32位（win32exts_exe.exe）和64位（win32exts_exe64.exe）的 进程外组件。

1、关于32位与64位的选择：
   64位运行时组件仅支持在x64系统上运行，32位组件可以同时在x86以及x64上运行。
   因此，同时考虑到 x64版本并未严格充分测试，建议一律使用 32位（x86）运行时组件。

2、关于进程内与进程外组件的选择：
   进程内（.dll）组件工作在 script.exe 解释器进程，运行效率较高，但会占用宿主进程内存。
   进程外（.exe）组件工作在独立进程，因此不占用宿主进程的内存空间，同时稳定性也更好。可以实现沙箱操作。
   可以根据实际需要选择之，不可一概而论。
